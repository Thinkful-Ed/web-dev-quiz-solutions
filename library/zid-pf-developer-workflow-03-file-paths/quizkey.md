## Developer workflow: File paths

1. Imagine you are in the `mammal/` directory of the following file structure.
    
    ```
    animal
    ├── mammal
    │   ├── bovine
    │   │   └── cow
    │   ├── canine
    │   └── equine
    │       ├── horse
    │       └── zebra
    └── reptile
        ├── bird
        ├── lizard
        │   └── salamander
        └── snake
    ```
    
    Which of the options is the correct path to the `cow/` directory?

    **Answer:** C. `./bovine/cow`
